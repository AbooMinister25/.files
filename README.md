# Hi, welcome to my Dotfiles :D

### Softwares Used

* WM: BSPWM
* bar: Polybar
* editor: NeoVim
* shortcuts: sxhkd
* shell: ZSH
* terminal: alacritty
* File Manager: Thunar
* Browser: Firefox
